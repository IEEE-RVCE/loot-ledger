## Setup

### 1. Install the required dependencies

```bash
$ yarn add
```

### 2. Rename the .env.example filename to .env and set your local variables

```bash
$ mv .env.example .env
```

### 3. Start the application

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Docker for development

```bash
# start the application
$ yarn docker:up

# stop the application
$ yarn docker:down
```
